Cole's Stop and Shop - Project
(Link to GitHub repo)

In the screenshots below you see the basic functionality of a cash register.

This was my first experience with dictionarys in C# and I learned a lot with this particular mimic project.

The idea with this project is that it's meant to mimic an API for a (imaginary) franchise
known as Cole's Stop and Shop.
Franchiees would hypothetically have this software implemented in their store when a new location is opened.

This console app first gets the user (cashier) to enter item id's which is meant to mimic the action of scanning bar codes on items.

After items are scanned and the cashier presses the key to finalize it prints the receipt
calculated with provincial and federal tax.
