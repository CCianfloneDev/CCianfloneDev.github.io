Mock Zoo Keeping System - Project
(Link to GitHub repo)

My idea with this project was to mimic a possible zoo keeping system
that would be used in a zoo from an management perspective.

There is 4 different menus. The main menu allows the user to enter either 3 different sub menus.
Employee management, equipment management, and animal management.

Each menu allows the administrator to add/remove attributes associated with the zoo.

As of Febuary 7th, 2022 I consider this to be my most complete personal-project.

Air C&C (mock-home-rental-system) - Project
(Link to GitHub repo)

This project sparked out of a joke with a peer. I decided to make a mimic program that would represent
something like AirB&B (for fun of course)

There will be a menu for Home owners to list their property, and a menu for renters to rent a property.
At this moment the only functionality I have implemented is just creating the property object
that will be accessed in both the renter and user interface.

Parking Garage (mock-parking-rental-system) - Project
(Link to GitHub repo)

This project is meant to mimic a terminal that cars would drive up to at the entrance of a parking garage,
choose their ticket option, and it would print a ticket associated with a stall number,
and repeat until parking lot is full.
